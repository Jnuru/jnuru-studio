FOUNDATION

Restraint with intent.

This page exists to test “alive but settled.” That means: fewer words, stronger hierarchy, and images doing the heavy lifting.

  • Type does the leadership.
  • Spacing is the personality.
  • Color is a whisper, not a banner.

TECHNICAL LAYOUT STUDY

Grid tension, controlled.

Slight offsets create motion without animation. The eye keeps moving even when nothing “moves.”

Rule

One loud thing per section.

Everything else supports it. No competing headlines.

Rule

Images must earn space.

If it’s decorative, cut it. If it explains you, keep it.

Rule

Accent stays rare.

Forest green appears as a detail, not a theme park.

IMAGE LANGUAGE

A “you” in pictures.

Pick 5 images that describe you without explaining you. Architecture. Instruments. Hands. Process. A calm portrait. Something unfinished.

NOTES

Rules that keep it calm.

No empty sections.

If it doesn’t say something, remove it or merge it.

One motion cue.

Offsets + spacing rhythm. Not animations everywhere.

Type scale stays consistent.

The “experiment” is layout + imagery, not random typography.
