FOUNDATION

Design before decoration.

The system should feel structured before it feels styled. Typography carries hierarchy, spacing creates rhythm, and color stays secondary.

  • Type leads the page.
  • Spacing does the emotional work.
  • Color supports, never dominates.

Brand voice

calm • precise • human
structure before styling
proof in the work, not the pitch

STRUCTURE

A document, not a landing page.

The page should read like a guideline. Each section repeats a clear pattern: title, explanation, and supporting context. That repetition creates trust.

Rule

Use a stable reading column.

Main content lives on the left. Supporting context sits on the right.

Anchor

One dominant move per section.

A strong title, a proof image, or a key rule. Not three things competing at once.

Rule

Section rhythm stays predictable.

Consistency makes the page feel authored and system-driven.

IMAGERY

Images should prove your way of working.

Use imagery to reveal process, materials, and authorship. A portfolio image should explain how you think, not just create mood.

Good

A real workspace with actual project work visible.

Good

A close crop of layout, type, code, or process material.

Avoid

Generic moodboard shots that say nothing about the work.

Image criteria

PRINCIPLES

Rules that protect the system.

The page should stay quiet, neutral, and disciplined. Once the system is working, visual personality can emerge from restraint instead of effects.

Every section must earn its place.

If a block adds no clarity, no proof, or no rhythm, remove it.

Motion comes from structure.

Use layout rhythm and spacing before reaching for animation.

Typography stays disciplined.

Variation should come from hierarchy and composition, not novelty.
