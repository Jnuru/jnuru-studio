CASE STUDY

Modernizing a dealership service center page

Overview: The Mazda Service Center page needed structural cleanup and visual modernization. The goal was to improve hierarchy, make the content easier to scan, and elevate the overall feel— without breaking dealership CMS constraints or SEO structure.

Role: UI/UX refinement + front-end implementation

Tools: HTML, CSS, Bootstrap grid patterns, dealership CMS components

The Problem

  • Visually heavy hero with weak hierarchy.
  • Inconsistent spacing and section rhythm.
  • Important information (hours, contact, value props) not clearly surfaced.
  • Content blocks felt stitched together instead of systemized.
  • Mobile experience lacked intention.

The Approach

Instead of redesigning from scratch, I restructured the existing layout into a cleaner, more modern system that could be reused across other service pages.

  • Hero simplification: Clear headline + supporting text + focused CTA.
  • Structured trust block: Hours, phone, and “why service with us” surfaced cleanly.
  • Service grid: Card-style layout for common services to improve scannability.
  • Amenities section: Clean chip-style presentation for quick value communication.
  • Facility imagery: Balanced image layout to add credibility and human presence.
  • Stronger closing CTA: Clear “Schedule Service” emphasis at both top and bottom.

Design Principles Applied

  • Clear visual hierarchy (headline → value → action).
  • Breathing room between sections.
  • Consistent button styling and CTA emphasis.
  • OEM-inspired cleanliness without overcomplicating layout.

Outcome

  • Cleaner, more modern presentation.
  • Improved readability and scan flow.
  • Stronger conversion emphasis on scheduling service.
  • Reusable section pattern for other dealership service pages.
